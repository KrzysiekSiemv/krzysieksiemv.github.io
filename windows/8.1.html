Windows 8.1

Windows 8.1 (pierwotnie oznaczony nazwą kodową Blue) – uaktualnienie systemu Windows 8 wydane przez firmę Microsoft, dostępne jako darmowa aktualizacja systemu, dla użytkowników posiadających Windows 8, aktualizacja dostępna w Sklepie Windows (ang. Windows Store). Dostępny również jako oddzielny system operacyjny, możliwy do kupienia i samodzielnej instalacji. Wikipedia

Jak aktywować system?

1. Uruchom wiersz polecenia (cmd) jako Administrator

2. Dodaj klucz do systemu za pomocą komendy: slmgr /ipk {KEY}

3. Ustaw serwer KMS: slmgr /skms kms8.msguides.com

4. Aktywuj swój system: slmgr /ato
