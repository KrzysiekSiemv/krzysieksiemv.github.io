Windows 8

Windows 8 – wersja systemu operacyjnego Microsoft Windows, produkowana przez Microsoft przeznaczona do użytku na komputerach osobistych, włączając w to domowe i firmowe komputery stacjonarne, laptopy i tablety PC. Proces rozwoju systemu rozpoczął się w 2009 roku przed premierą jego poprzednika Windows 7. Jego istnienie zostało potwierdzone na targach CES 2011, gdzie zaprezentowano wersję działającą na procesorze ARM, następnie od września 2011 do maja 2012 wydano trzy wersje rozwojowe systemu. 27 sierpnia 2012 system został uznany za ukończony i uzyskał status RTM. Oficjalna premiera systemu Windows 8 odbyła się 26 października 2012 roku w Nowym Jorku. Razem z nim Microsoft zaprezentował system Windows Phone 8, przeznaczony na telefony komórkowe. Wikipedia

Jak aktywować system?

1. Uruchom wiersz polecenia (cmd) jako Administrator

2. Dodaj klucz do systemu za pomocą komendy: slmgr /ipk {KEY}

3. Ustaw serwer KMS: slmgr /skms kms8.msguides.com

4. Aktywuj swój system: slmgr /ato
