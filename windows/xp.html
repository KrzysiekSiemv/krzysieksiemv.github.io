Windows XP

Windows XP (nazwa kodowa Whistler) – wersja systemu operacyjnego Microsoft Windows z rodziny Windows NT (oparty na jądrze NT) firmy Microsoft, wydana 25 października 2001 roku. Jednocześnie w tym dniu wydano Windows Media Player dla Windows XP. Koniec wsparcia technicznego dla tego systemu operacyjnego nastąpił 8 kwietnia 2014, natomiast wersja POSReady 2009 była wspierana do 9 kwietnia 2019 roku. Wikipedia
