Windows Vista

Windows Vista (Windows NT 6.0, nazwa kodowa Longhorn) – system operacyjny z rodziny Windows wydany przez Microsoft 30 stycznia 2007 roku. Jest następcą systemu Windows XP. Kolejnym systemem Microsoftu jest Windows 7 wydany w 2009 roku. Wsparcie od Microsoft zakończyło się 11 kwietnia 2017 roku. Wikipedia
