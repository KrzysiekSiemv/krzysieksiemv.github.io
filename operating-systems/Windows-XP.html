Windows XP

Wersja systemu operacyjnego Microsoft Windows z rodziny Windows NT (oparty na jądrze NT) firmy Microsoft, wydana 25 października 2001 roku. Jednocześnie w tym dniu wydano Windows Media Player dla Windows XP

Wymagania sprzętowe

Procesor procesor o częstotliwości 300 MHz lub wyższej
Pamięć RAM 128 MB
Miejsce na dysku twardym 2 GB
Karta graficzna Adapter wideo i monitor o wyższej rozdzielczości
Wyświetlacz 800 x 600
Połączenie internetowe Połączenie z Internetem jest niezbędne do przeprowadzenia aktualizacji, pobierania i korzystania z niektórych funkcji.
