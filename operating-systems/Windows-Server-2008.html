Windows Server 2008

Serwerowy system operacyjny opracowywany przez firmę Microsoft. Do 15 maja 2007 był znany jedynie pod nazwą kodową Windows Server „Longhorn”. Windows Server 2008 R2 jest następcą systemu Windows Server 2008. Oficjalna premiera odbyła się 22 października 2009

Wymagania sprzętowe

Procesor procesor o częstotliwości 1 GHz dla wersji 32-bitowej lub 1.4 GHz dla wersji 64-bitowy
Pamięć RAM 512 MB lub 2 GB dla z środowiskiem pulpitowym
Miejsce na dysku twardym 10 GB
Karta graficzna biblioteki DirectX 9 lub nowsze ze sterownikiem WDDM 1.0
Wyświetlacz 800 x 600
Połączenie internetowe Połączenie z Internetem jest niezbędne do przeprowadzenia aktualizacji, pobierania i korzystania z niektórych funkcji.
