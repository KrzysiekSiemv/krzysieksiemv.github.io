Windows Server 2016

Serwerowy system operacyjny, tworzony przez Microsoft z rodziny Windows NT, rozwijany równolegle z systemem Windows 10. Pierwsza wersja testowa pojawiła się już 1 października 2014 roku

Wymagania sprzętowe

Procesor procesor o częstotliwości 1.4 GHz lub wyższej 64-bitowy
Pamięć RAM 512 MB lub 2 GB dla z środowiskiem pulpitowym
Miejsce na dysku twardym 32 GB
Karta graficzna biblioteki DirectX 9 lub nowsze ze sterownikiem WDDM 1.0
Wyświetlacz 1024 x 768
Połączenie internetowe Połączenie z Internetem jest niezbędne do przeprowadzenia aktualizacji, pobierania i korzystania z niektórych funkcji.

Jak aktywować system?

Wystarczą trzy komendy w Wierszu Polecenia. Oczywiście potrzeba uprawnień administratora. Wpisujemy kolejno komendy:

slmgr /ipk {KLUCZ} | slmgr /skms kms8.msguides.com | slmgr /ato
