Windows Server 2012

Windows Server 2012 (nazwa kodowa "Windows Server 8") – szóste wydanie Windows Server, wersja serwerowa systemu Windows 8 i następca Windows Server 2008 R2. Dwie wersje developerskie zostały udostępnione w fazie testów. System był dostępny do kupienia 4 września 2012 roku. Odmiennie od swoich poprzedników Windows Server 2012 nie wspiera komputerów opartych o Itanium i posiada cztery edycje. Wiele właściwości zostało dodanych bądź poprawionych w porównaniu do Windows Server 2008 R2, jak na przykład zaktualizowana wersja Hyper-V, nowa wersja menedżera zadań oraz system plików ReFS. Dodatkowo system posiada interfejs Modern UI. Wikipedia

Jak aktywować system?

1. Uruchom wiersz polecenia (cmd) jako Administrator

2. Dodaj klucz do systemu za pomocą komendy: slmgr /ipk {KEY}

3. Ustaw serwer KMS: slmgr /skms kms8.msguides.com

4. Aktywuj swój system: slmgr /ato

Windows Server 2012 R2

Windows Server 2012 R2 is the sixth version of the Windows Server server operating system by Microsoft, as part of the Windows NT family of operating systems. It was unveiled on June 3, 2013 at TechEd North America, and released on October 18, 2013. Wikipedia
